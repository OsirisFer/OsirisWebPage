Componentes de React

¿Qué es un Componente?

Un componente en React es una pieza reutilizable de la interfaz de usuario que puede tener su propio estado y lógica. Los componentes se pueden anidar, gestionar y manejar independientemente.

Componentes Funcionales son funciones de JavaScript que aceptan un único argumento (props) y devuelven elementos de React.


function ComponenteFuncional(props) {
  return <h1>Hola, {props.nombre}!</h1>;
}

Componentes de Clase Son clases de ES6 que extienden de React.Component y tienen un método render() que devuelve elementos de React.


class ComponenteClase extends React.Component {
  render() {
    return <h1>Hola, {this.props.nombre}!</h1>;
  }
}

Propiedades (Props)Las propiedades son argumentos que se pasan a los componentes y se utilizan para pasar datos e invocar funciones de un componente a otro.


function ComponenteConProps(props) {
  return <p>Mi nombre es {props.nombre} y tengo {props.edad} años.</p>;
}

// Uso del componente
<ComponenteConProps nombre="Juan" edad={30} />

Estado (State)El estado es un objeto que determina cómo se renderiza y se comporta un componente. Es gestionado dentro del componente y puede cambiar con el tiempo.


class ComponenteConEstado extends React.Component {
  constructor(props) {
    super(props);
    this.state = { contador: 0 };
  }

  incrementar = () => {
    this.setState({ contador: this.state.contador + 1 });
  }

  render() {
    return (
      <div>
        <p>Contador: {this.state.contador}</p>
        <button onClick={this.incrementar}>Incrementar</button>
      </div>
    );
  }
}

Ciclo de Vida Los componentes de clase tienen métodos del ciclo de vida que se invocan en diferentes etapas del ciclo de vida del componente, como montaje, actualización y desmontaje.


class ComponenteCicloDeVida extends React.Component {
  componentDidMount() {
    // Código que se ejecuta después de que el componente se ha montado
  }

  componentDidUpdate(prevProps, prevState) {
    // Código que se ejecuta después de que el componente se ha actualizado
  }

  componentWillUnmount() {
    // Código que se ejecuta justo antes de que el componente se desmonte
  }

  render() {
    return <div>Componente con ciclo de vida</div>;
  }
}
